Pan and Zoom

Click and drag on an empty area of the diagram to move around.

Zoom in and out using your mouse scrollwheel. Alternatively, if you have a touchscreen or trackpad, use the standard pinch zoom gestures.

Nodes

The Process Model is a directed acyclic graph. It is formed of nodes connected to other nodes through edges. The leftmost node is called the root. All nodes without any edges coming out of them are called leaves.

To the right of most nodes (some cannot have children) is a circle, which we call the junction. Left click on this to create a new node.

To create a new edge between to existing nodes, left-click and drag from a junction onto another node. This may not always be allowed, as there are restrictions the kinds of nodes which may be connected.

At the end of each edge is a smaller circle. Left click on this to disconnect that edge. Disconnecting all edges from a node will remove it.

Name and Description

At the top of each node is its name. Left-click on it to edit it, but note that each name must be unique. A name may also be a URL - if it begins with 'http' or 'https', then a link will be provided.

Underneath the name is a description of the node. Again, click to edit.

Layout

By default, nodes and edges are layed out automatically left-to-right.

Left-click and drag on a node to position it manually. Left-click and drag on the resize handle at the bottom-right of each node to resize it.

Right-click on a manually positioned or sized node to return it to automatic layout with the default size.
