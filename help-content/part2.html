Import and Export

You may save your process model by clicking on the download link. You can reload the file produced by dragging and dropping it back onto the page.

The model has support for importing xml files exported from PeriMeta. Again, drag and drop the file onto the page.

Html Import

The model may also be loaded from a set of linked html pages. Embed process-model/index.html as an iframe inside the page which is to act as the root node. The following properties will be scraped:

  • The URL of the page becomes the name of the node.
  • A table containing columns named 'evidence', 'necessity' and 'sufficiency' is a list of edges. The evidence column should contain an HTML link.
  • A table containing the column 'dependence' will be used to set the dependence value for a node which is not a leaf node (has no children).
  • A table containing the columns 'success' and 'failure' will be used to set the local evidence for a leaf node.

Note that you will either need to serve all pages from the same origin, or add the Access-Control-Allow-Origin header to the parent page.
